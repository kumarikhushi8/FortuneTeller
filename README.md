# 🔮 Fortune Teller

A fun beginner-friendly JavaScript project that generates a random fortune using variables, conditional statements, and the `Math.random()` method.

## ✨ Features

- Generates random fortunes
- Uses `Math.random()` for randomness
- Demonstrates conditional statements
- Beginner-friendly JavaScript practice project
- Displays a different fortune each time the program runs

## 🛠️ Technologies Used

- JavaScript

## 📜 Example Output

```bash
You will find a new hobby soon.
```

```bash
Be cautious of your new neighbors.
```

## 🚀 How to Run

1. Copy the code into a file named `script.js`
2. Open terminal in the project folder
3. Run:

```bash
node script.js
```

## 📚 Concepts Practiced

- Variables (`let`)
- Random number generation
- `Math.random()`
- Conditional statements (`if...else`)
- Console output

## 🎯 Purpose

This project was created as a beginner JavaScript exercise to practice randomness, conditional logic, and dynamic program behavior in a fun way.
